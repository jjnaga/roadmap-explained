HTML

<!doctype html> <html lang="en"> <head> <meta charset="utf-8"> <title></title> <link rel="stylesheet" href="stylesheet.css"> </head> <body> </body> </html>

Above is the bare minimum of what consitutes an HTML page. You can argue that HTML requires less or more, but for the sake of a greenhorn web developer, this is all you need.


<!doctype html>

As described in Ch0, HTML is just text. Through tags such as:

<p></p> the browser knows that this line of text is a pargraph, or a header, or a link. By itself, any html file is nothing more than text. However, when a web browser opens that file, it takes all your tags, and implements what it thinks a tag should look like.
